createexcel Examplesâ€‹

Your excel_template.xlsx has to be included in the FormDefinition template files and should look something like this: Excel Template

After filling out the fields in your Form and completing the stage, an Excel file will be created with the name you provided (in our case, form.xlsx), containing the filled values. You can find this file in the Admin UI under your Form Settings, Media & Attachments < Others. Excel Template - Filled
